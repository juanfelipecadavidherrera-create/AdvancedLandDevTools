Advanced Land Development Tools v1.0

A productivity suite for Autodesk Civil 3D 2025/2026 designed for land development professionals. All commands are accessible from the Advanced Land Dev Tools ribbon tab or the floating Mini Toolbar.

Installation

The plugin installs automatically via the Autodesk App Store. After installation, restart Civil 3D. The Advanced Land Dev Tools ribbon tab will appear automatically.

No manual DLL loading or NETLOAD commands are required.

Licensing

The plugin includes a 14-day free trial starting from the first use. During the trial, all commands are fully functional.

After the trial period, purchase a license from the Autodesk App Store to continue using the plugin. The license is verified automatically through your Autodesk account.

Offline use: If you have a valid license and lose internet connectivity, the plugin will continue working for up to 3 days before requiring a connection to re-verify.

Commands

Profile Views

BULKSUR — Bulk Surface Profile Creator
Creating surface profiles one alignment at a time is one of the most repetitive tasks in land development. BULKSUR eliminates that bottleneck entirely. Select any number of alignments in your drawing, configure your station range, elevation range, and profile view style, then pick a single insertion point. The tool generates every surface profile and profile view in one batch, automatically spacing them vertically so they don't overlap. What normally takes 20–30 minutes of clicking through dialogs is reduced to a few seconds.
Workflow: Select alignments → configure settings in dialog → pick insertion point → all profile views are created and stacked automatically.
GETPARENT — Get Parent Alignment
When working in a drawing with dozens of profile views, it's easy to lose track of which alignment a particular profile view belongs to. GETPARENT solves this instantly. Click any profile view and the tool identifies its parent alignment, highlights it in the drawing, and reports its name and station range in the command line. This is especially useful in large subdivision projects where profile views may have been moved or regrouped far from their source alignments.
Workflow: Click any profile view → parent alignment is selected and its name and station range are reported.
PIPEMAGIC — Pipe Crossing Projection
Projecting pipe networks into profile views is a critical but tedious step in utility coordination. PIPEMAGIC automates it completely. Select one or more profile views and the tool scans all pipe networks in the drawing — both gravity and pressure — detects which ones cross each alignment, and projects them into the corresponding profile view automatically. No need to manually identify crossings or run the projection dialog for each network one by one. This is invaluable for drainage reviews, utility conflict detection, and construction plan preparation.
Workflow: Select one or more profile views → all crossing pipe networks (gravity and pressure) are detected and projected automatically.
MARKLINES — Mark Crossing Lines
Utility lines, right-of-way boundaries, easements, and other linear features drawn as lines or polylines in plan view often need to be referenced in profile views. MARKLINES finds every point where selected lines or polylines cross an alignment and draws a full-height vertical marker line at that station in the profile view. This gives you instant visual reference for where utilities or boundaries cross your road or corridor — without manually measuring stations or placing lines by hand.
Workflow: Select layers containing lines/polylines → pick a profile view → vertical markers are drawn at every crossing station.

Alignments

ALIGNDEPLOY — Alignment Deploy
In subdivision and site layout work, you often need to place the same cross alignment (like a driveway, cul-de-sac, or side street template) at regular intervals along a main road alignment. ALIGNDEPLOY automates this repetitive placement. Select your main alignment and a cross alignment, specify the interval, and the tool places rotated copies of the cross alignment at each station along the main alignment, properly oriented perpendicular to the road centerline. This dramatically speeds up preliminary layout for lot access, drainage crossings, or any repeating cross-pattern.
Workflow: Select main alignment → select cross alignment → enter interval → copies are deployed along the main alignment at the specified spacing.

Pipes

INVERTPULLUP — Invert Pull Up
During construction plan reviews and field coordination, engineers frequently need to know the exact invert elevation at a specific point along a pipe — not just at the structures. INVERTPULLUP makes this a one-click operation. Click any gravity or pressure pipe, then click the point along it where you need the invert. The tool calculates the precise elevation using linear interpolation between the pipe's start and end inverts, then places a station-offset label directly in the drawing with the calculated value. This eliminates manual slope calculations and reduces the risk of field errors.
Workflow: Click a pipe → click a point on that pipe → invert elevation is calculated and a station-offset label is placed automatically.
CHANGEELEVATION — Change Pipe Elevation
When designing or adjusting pipe networks, you sometimes need to make a pipe segment perfectly level — matching both ends to the same invert elevation. CHANGEELEVATION simplifies this. Select any gravity or pressure pipe and the tool displays both the start and end invert elevations. Choose which elevation to apply, and the tool sets both ends to that value, making the pipe level. This is particularly useful for adjusting stub-outs, lateral connections, or correcting pipe segments during design iterations.
Workflow: Select a pipe → view start and end inverts → choose which elevation to apply to both ends → pipe is leveled.

Property Information

FLOODZONE — FEMA Flood Zone Lookup
Knowing the FEMA flood zone designation for a property is one of the first things required in any land development project. It determines building elevation requirements, insurance obligations, and permitting conditions. FLOODZONE queries the FEMA National Flood Hazard Layer (NFHL) in real time based on a point you pick in the drawing. It returns the flood zone designation (e.g., AE, X, AH), base flood elevation (BFE), whether the location is in a Special Flood Hazard Area (SFHA), and the FIRM panel number. You can optionally place the results as an MText label directly in the drawing for reference.
Workflow: Pick a point in the drawing → FEMA data is queried live → flood zone, BFE, SFHA status, and FIRM panel are reported → optionally place a label.
FLOODCRITERIA — County Flood Criteria
In Miami-Dade County, development permits require that finished floor elevations meet the county's Flood Criteria — a dataset separate from FEMA maps. FLOODCRITERIA queries the Miami-Dade County Flood Criteria 2022 dataset and reports the minimum finished floor elevation (in ft NGVD) for the location you pick. This saves a trip to the county's GIS portal and ensures you have the correct regulatory elevation early in design.
Workflow: Pick a point in the drawing → Miami-Dade County Flood Criteria is queried → minimum finished floor elevation is reported.
SECTIONLOOKUP — PLSS Section Lookup
Legal descriptions and permit applications in Florida frequently require the PLSS (Public Land Survey System) section, township, and range. SECTIONLOOKUP queries the South Florida Water Management District (SFWMD) PLSS service and returns the Section number, Township, Range, and the combined SSTTRR code for any point you pick in the drawing. This eliminates the need to look up this information on separate GIS websites or paper maps.
Workflow: Pick a point in the drawing → SFWMD PLSS service is queried → Section, Township, Range, and SSTTRR code are reported.
GROUNDWATER — Water Table Lookup
Groundwater elevation is a critical design parameter for stormwater management, foundation design, and exfiltration trench sizing. GROUNDWATER queries the Miami-Dade County Groundwater Level dataset (based on USGS model projections) and returns the water table elevation at the picked point in both NAVD 88 and NGVD 29 datums. Having both values readily available avoids datum conversion errors that can lead to costly design mistakes.
Workflow: Pick a point in the drawing → groundwater level is queried → elevation is reported in both NAVD 88 and NGVD 29.

Areas

AREAMANAGER — Area Manager
Tracking areas across a land development project — impervious surfaces, pervious areas, lakes, building footprints — is essential for stormwater calculations, zoning compliance, and permit applications. The Area Manager is a dockable palette (similar to Toolspace) that stays open while you work. Create a project, then add areas by selecting hatches or closed polylines in your drawing. Each area is stored with its name, square footage, layer, and boundary coordinates. You can organize areas into custom categories (e.g., "Impervious", "Pervious", "Lake") and rename them at any time. If a hatch is deleted from the drawing, the data is preserved — click Redraw to recreate it at its original location and zoom to it. Export a complete area report to CSV with per-category subtotals for easy integration into permit applications and engineering calculations. All project data is saved locally and persists across Civil 3D sessions.
Workflow: Type AREAMANAGER → create/select a project → click Add Area → select a hatch or closed boundary → area is stored and displayed → rename, categorize, redraw, or export as needed.

Quick Access

ALDTTOOLBAR — Toggle Mini Toolbar
The Mini Toolbar is a compact floating panel that gives you one-click access to every ALDT command without switching ribbon tabs. It stays on top of the drawing area, so you can keep working in any Civil 3D ribbon tab (Home, Modify, Analyze, etc.) and still have all ALDT tools within reach. Drag the toolbar anywhere on screen to reposition it. Click the X button to close it, or type ALDTTOOLBAR again to toggle it back. The toolbar is organized by category — Profiles, Alignments, Pipes, Info, and Layout — with color-coded icons for fast visual identification.
Workflow: Type ALDTTOOLBAR or click the Mini Toolbar button on the ribbon → toolbar appears → click any icon to run the command → drag to reposition.
