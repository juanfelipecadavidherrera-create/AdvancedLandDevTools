Advanced Land Development Tools v1.1.0

A productivity suite for Autodesk Civil 3D 2025 / 2026 / 2027 designed for land development professionals. All commands are accessible from the Advanced Land Dev Tools ribbon tab or the floating Mini Toolbar.

Use the sidebar on the left to jump directly to any command.

Installation

The plugin installs automatically via the Autodesk App Store. After installation, restart Civil 3D. The Advanced Land Dev Tools ribbon tab will appear automatically.

No manual DLL loading or NETLOAD commands are required.

Licensing

The plugin includes a 30-day free trial starting from the first use. During the trial, all commands are fully functional.

After the trial period, purchase a license from the Autodesk App Store to continue using the plugin. The license is verified automatically through your Autodesk account -- no manual activation is required.

Offline use: If you have a valid license and lose internet connectivity, the plugin will continue working for up to 3 days before requiring a connection to re-verify.

Subscription renewal: If your subscription lapses and is later renewed, the plugin restores full access automatically on the next license check -- no reinstallation needed.

Commands

Profile Views

BULKSUR -- Bulk Surface Profile Creator
Creating surface profiles one alignment at a time is one of the most repetitive tasks in land development. BULKSUR eliminates that bottleneck entirely. Select any number of alignments, configure your station range, elevation range, and profile view style, then pick a single insertion point. The tool generates every surface profile and profile view in one batch, automatically spacing them vertically so they do not overlap.
How to use:
  1. Run BULKSUR from the ribbon or command line.
  2. In the dialog, select which alignments to process and configure profile view style, station range, and elevation range.
  3. Click OK, then pick an insertion point in model space.
  4. All profile views are created and stacked automatically at the chosen point.
GETPARENT -- Get Parent Alignment
Click any profile view and the tool identifies its parent alignment, highlights it in the drawing, and reports its name and station range in the command line. Useful in large subdivision projects where profile views may have been moved far from their source alignments.
How to use:
  1. Run GETPARENT.
  2. Click any profile view in the drawing.
  3. The parent alignment is selected and its name and station range are reported.
PIPEMAGIC -- Pipe Crossing Projection
Select one or more profile views and PIPEMAGIC scans all pipe networks in the drawing -- both gravity and pressure -- detects which ones cross each alignment, and projects them into the corresponding profile view automatically.

Pressure Fitting Detector -- When enabled, also projects the start and end fittings of each crossing pressure pipe into the profile view. Activate by typing On at the prompt.

Manhole Mode -- When enabled (mutually exclusive with Fitting Detector), projects the start and end structures of each crossing gravity pipe. You can choose which gravity networks to include.
How to use:
  1. Run PIPEMAGIC.
  2. Select one or more profile views (press Enter when done).
  3. At the Pressure Fitting Detector [On/Off] prompt, type On or Off.
  4. If Fitting Detector is Off, you are prompted for Manhole Mode [On/Off].
  5. If Manhole Mode is On, select which gravity networks to include from the list.
  6. All crossing pipes, fittings, and/or structures are projected automatically.
LLABELGEN -- Label Generator v1.1.0
Automatically places 3D Profile View Station Elevation Labels at crossing pipe inverts. Select a profile view (native or XREF), the tool detects all crossing pipes and pressure pipes, filters by network, allows you to choose your label and marker styles, and places the formatted Station Elevation Labels exactly at the pipe inverts.
How to use:
  1. Run LLABELGEN from the ribbon, mini toolbar, or command line.
  2. Click on a profile view in the drawing (works for both native and XREF profile views).
  3. Select a network to exclude, or enter 0 to keep all.
  4. Choose your desired Station Elevation Label Style and Marker Style from the dialog.
  5. The tool automatically generates the 3D labels at all pipe crossing inverts.
MARKFITTINGS -- Mark Pressure Fittings v1.1.0
Draws dashed vertical marker lines in a profile view for pressure fittings. Select a profile view, and the tool automatically detects pressure fittings within it and draws dashed vertical marker lines from top to bottom.
How to use:
  1. Run MARKFITTINGS.
  2. Select a profile view in the drawing.
  3. The tool will automatically detect pressure fittings and draw dashed vertical lines.
MARKLINES -- Mark Crossing Lines
Finds every point where selected lines or polylines cross an alignment and draws a full-height vertical marker line at that station in the profile view. Multiple profile views can be processed in a single run.
How to use:
  1. Run MARKLINES.
  2. In the layer selection dialog, choose the layers containing lines or polylines to scan.
  3. Pick profile views one by one in the drawing (press Enter when done).
  4. Vertical marker lines are drawn at every crossing station across all selected views.
OFFPV -- Turn Off Network Parts in a Profile View v1.1.0
Turn off network parts in a profile view -- keep selected pipes. Click any pipe or structure drawn in a profile view to identify the network and the profile view, then select the parts you want to keep visible. Every other part of the same network in that profile view is removed via Civil 3D's RemoveFromProfileView. Works on both gravity networks (pipes + structures) and pressure networks (pipes + fittings + appurtenances). Affects only the picked profile view -- other profile views that show the same network are untouched.
How to use:
  1. Run OFFPV.
  2. Click any pipe or structure drawn in a profile view to identify the network and the profile view.
  3. Select (window/pick) the parts you want to KEEP visible, then press Enter to finish.
  4. Pressing Enter without selecting anything turns OFF the entire network in that profile view.
  5. Every other part of the same network in that profile view is removed automatically.
Note: Not reversible by the plugin -- use AutoCAD undo, or manually re-add parts to the profile view through Civil 3D.
PROFOFF -- Profile Off v1.1.0
Remove a specific pipe or structure from a profile view by clicking directly on it. The part is removed from that profile view only -- it remains in the model and is still shown in all other views.
How to use:
  1. Run PROFOFF.
  2. Click on the profile view border or background to identify the view (Step 1).
  3. Click on each pipe or structure crossing symbol to hide it (Step 2).
  4. Repeat for as many parts as needed. Press Enter or Escape to finish.
Tip: Click the crossing circle or line symbol for the pipe inside the profile view. Works with gravity pipes, structures, pressure pipes, pressure fittings, and pressure appurtenances.
PVSTYLE -- Profile View Style Override v1.1.0
Changes the per-profile-view style override for a pipe or structure -- the "Style Override" column in Profile View Properties. The global pipe style is never modified; only the display inside the selected profile view changes.
How to use:
  1. Run PVSTYLE.
  2. Click on the profile view border or background to lock the view (Step 1).
  3. Click on a pipe or structure symbol inside the profile view (Step 2).
  4. A numbered list of all available pipe or structure styles appears in the command window.
  5. Type the number of the style you want and press Enter.
  6. Repeat for additional parts. Press Enter or Escape to finish.
Tip: The style list includes every pipe/structure style defined in the drawing, not just those currently in use -- so styles like "SEWER CROSSING ONLY" will appear even if no pipe is currently using them globally.
CHOPCHOP -- Profile View Subdivider v1.1.0
Subdivide an existing profile view into multiple smaller profile views along the station axis. The original profile view is preserved untouched. Two subdivision modes are available:

Equal Subdivision -- enter a segment length (e.g. 500 ft) and the tool divides the station range into equal segments. If the total length is not evenly divisible, the last segment absorbs the remainder.

Custom Intervals -- manually define start and end stations for each sub-view. The tool validates for overlaps and out-of-range values.

Sub-views are placed 500 ft below the original (configurable), arranged horizontally left to right in station order with a configurable gap between them. Styles, elevation ranges, and band configurations are copied from the source profile view.
How to use:
  1. Run CHOPCHOP from the ribbon, mini toolbar, or command line.
  2. Click on a profile view in the drawing to select it as the source.
  3. In the dialog, choose Equal Subdivision or Custom Intervals.
  4. For Equal: enter the segment length in feet (e.g. 500). A live preview shows how many sub-views will be created and their station ranges.
  5. For Custom: add rows with start/end stations for each interval. Use station format (e.g. 5+00) or raw numbers (e.g. 500).
  6. Adjust the Vertical Offset (default 500 ft) and Horizontal Gap (default 100 ft) if needed.
  7. Click OK. Sub-views are created and arranged below the original.
Tip: The original profile view is never modified or deleted. Sub-views inherit the same profile view style and elevation range from the source.
VPCUT -- Viewport Clip Layout
Clips a paper space viewport to an arbitrary closed shape drawn in model space -- perfect for lot-shaped viewports, curved road boundaries, or any non-rectangular presentation frame.

Supported boundary types: circles, closed polylines, rectangles, splines, and any other closed curve.
How to use:
  1. Switch to a paper space layout.
  2. Run VPCUT and select the viewport to clip.
  3. Switch into model space and select the closed shape that defines the clip boundary.
  4. The viewport is clipped to that shape instantly.

Alignments

ALIGNDEPLOY -- Alignment Deploy
Places rotated copies of a cross alignment at regular intervals along a main alignment, properly oriented perpendicular to the road centerline. Ideal for driveways, cul-de-sacs, drainage crossings, or any repeating cross-pattern.
How to use:
  1. Run ALIGNDEPLOY.
  2. Select the main (long) alignment.
  3. Select the cross (short) alignment to copy.
  4. Enter the deployment interval in drawing units.
  5. Copies are placed along the main alignment at the specified spacing.

Pipes

INVERTPULLUP -- Invert Pull Up
Click any gravity or pressure pipe, then click a point along it. The tool calculates the exact invert elevation via linear interpolation and places a station-offset label at that location. Eliminates manual slope calculations.
How to use:
  1. Run INVERTPULLUP.
  2. Click on a gravity or pressure pipe.
  3. Click a point along that pipe where you need the invert elevation.
  4. The invert is calculated and a station-offset label is placed automatically.
CHANGEELEVATION -- Change Pipe Elevation
Select a gravity or pressure pipe to display both crown elevations, then choose which one to apply to both ends, making the pipe level. Works by selecting the pipe directly in plan view or by clicking the pipe crossing symbol inside a profile view.
How to use:
  1. Run CHANGEELEVATION.
  2. Click on a pipe in plan view, or click a pipe crossing symbol inside a profile view.
  3. The start and end outside crown elevations are displayed.
  4. Type 1 to use the start elevation or 2 to use the end elevation.
  5. Both ends of the pipe are set to the chosen elevation.
PIPESIZING -- Pipe Sizing Calculator
Opens a calculator window for stormwater pipe sizing. Enter the drainage area, runoff coefficient, and design storm intensity; then check a specific pipe diameter and slope against Manning's Equation capacity and velocity criteria.
How to use:
  1. Run PIPESIZING.
  2. Enter the drainage area (acres), runoff coefficient C, and rainfall intensity (in/hr).
  3. Enter the pipe diameter and slope to check.
  4. The tool calculates required flow (Q = CiA), pipe capacity, and velocity, and reports pass/fail.
EEEBEND -- EEE Bend (Pressure Pipe Duck)
Automates the placement of a pressure pipe bypass ("duck") in a profile view. Select the pressure pipe run, click the crossing location, enter the target invert elevation -- the command places all four bends and five pipe segments automatically.
How to use:
  1. Run EEEBEND.
  2. Select a pressure pipe run in a profile view.
  3. Click the crossing location along the pipe.
  4. Enter the invert elevation at the crossing low point.
  5. Four bends and five pipe segments are placed creating the duck profile automatically.
LATERALBEAST -- Lateral Pipe Drafter v1.1.0
Automatically drafts lateral pipes in Civil 3D profile views based on user-selected networks and target layers. Calculates precise geometric offsets from pipe inverts and projects the lateral line until it intersects with the chosen target layer.
How to use:
  1. Run LATERALBEAST.
  2. Select one or more profile views.
  3. Select the pipe network the laterals belong to.
  4. Choose the target line layer (e.g., P-ROW) from the dialog, or access via the Settings keyword.
  5. The command will automatically find the inverts and draw the laterals based on the specified direction, angle, and gap.
COVERADJUST -- Pressure Duck Cover Adjust v1.1.0
Moves both PVIs of a pressure duck (4 bends) vertically to adjust cover. Select a pressure pipe duck inside a profile view. The tool auto-detects the ground-surface elevation from the alignment's surface profile, computes the minimum depth that gives both fittings at least 4 ft of crown cover, and moves both PVIs to that elevation so they sit at the same height.
How to use:
  1. Run COVERADJUST.
  2. Select a pressure pipe duck in a profile view.
  3. The PVIs are adjusted to maintain minimum cover automatically.
PRESSCOUNT -- Pressure Network Count
Scans the drawing, lets you select a pressure network, calculates total 3D pipe length, assigns sequential numbers to every fitting, and optionally places MText labels at each fitting location in plan view. All labels are grouped into a single named AutoCAD Group.
How to use:
  1. Run PRESSCOUNT.
  2. In the dialog, select the pressure network to analyze.
  3. Click OK. Total 3D pipe length and fitting count are reported.
  4. Choose whether to place MText labels at each fitting location.
  5. If yes, enter the desired text height. Labels are placed and grouped automatically.
RRNETWORKCHECK -- Pressure Network Clearance Check BETA v1.1.0
Checks vertical clearance and cover between pressure pipe parts and the existing ground surface in a profile view. Click any pressure pipe part shown in a profile view and the tool auto-detects the surface sampled on that view, then measures the distance from the pipe to the surface at every relevant point. Reports pass/fail based on minimum cover requirements. Uses logic to suggest bend placements when clearance is insufficient.

Surface Auto-Detection -- The tool reads the profile view's sampled surface profiles and automatically identifies the source surface via the Civil 3D DataSourceId API. No manual surface selection is needed.

Note: This command is currently in beta. Bend placement suggestions are preliminary and should be reviewed before construction.
How to use:
  1. Run RRNETWORKCHECK from the ribbon, mini toolbar, or command line.
  2. Click on a pressure pipe part (pipe, fitting, or appurtenance) inside a profile view.
  3. The tool auto-detects the surface from the profile view's sampled profiles.
  4. Cover and clearance values are calculated at every significant point along the pipe.
  5. A pass/fail report is shown in both the command line and a summary dialog.
  6. If clearance violations are found, the tool suggests bend locations based on the surface profile.
Tip: Make sure the profile view has at least one surface profile sampled (EG/Existing Ground) so the tool can auto-detect the reference surface. If no surface profile is found, you will be prompted to select one manually.
LATMANAGER -- Lateral Crossing Manager v1.1.0
A dockable palette for tracking and projecting 2D sewer laterals across profile views. Since 2D laterals lack 3D information, users can draw an ellipse representing the pipe in the lateral's profile view. LATMANAGER saves that crossing data (station and invert elevation) and allows you to project it accurately as an ellipse into any intersecting profile view (e.g., a water main profile).
How to use:
  1. Run LATMANAGER to open the Lateral Manager palette.
  2. Create or load a project.
  3. Click Add Crossing Info, select a lateral's profile view, and then click the ellipse representing the pipe crossing.
  4. To project saved laterals, click Project to Profile, and select a target profile view.
  5. The tool automatically checks for alignment intersections and draws ellipses at the correct stations and saved elevations.

Surfaces

ELEVSLOPE -- Elevation Sloper
Pick a starting point on a TIN surface, enter a slope and distance, then click to place a new surface point at the calculated elevation. Ideal for projecting elevations along a designed grade.
How to use:
  1. Run ELEVSLOPE.
  2. Pick a starting point on a TIN surface.
  3. Enter the slope and direction.
  4. A new surface point is added at the calculated elevation.
BLOCKTOSURFACE -- Block to Surface
Reads a specified elevation attribute from selected blocks and adds their X, Y locations and attribute-derived elevations as breakpoints to a Civil 3D TIN surface. Useful for incorporating survey point data delivered as AutoCAD blocks.
How to use:
  1. Run BLOCKTOSURFACE.
  2. Select the blocks containing elevation attributes.
  3. Specify the attribute tag name that holds the elevation value.
  4. Select the target TIN surface. Block locations are added as point data.
TEXTTOSURFACE -- Text to Surface
Reads the numeric content of selected Text or MText entities and adds each entity's location to a Civil 3D TIN surface as a point. Useful for converting spot elevation annotations into usable surface data.
How to use:
  1. Run TEXTTOSURFACE.
  2. Select the text or MText entities containing elevation values.
  3. Select the target TIN surface. Each text location and parsed elevation are added.
LOWRIM -- Lowest Rim Elevation
Scans selected Civil 3D structures and identifies the one with the lowest rim elevation -- typically the controlling structure for a drainage system or the first to overflow.
How to use:
  1. Run LOWRIM.
  2. Select the Civil 3D structures to check.
  3. The structure with the lowest rim elevation is highlighted and its name and elevation are reported.

Property Information

FOLIO -- Property Appraiser Lookup v1.1.0
Queries the Miami-Dade County Property Appraiser (MDC GIS) to retrieve property information including folio, owner name, and address.
How to use:
  1. Run FOLIO.
  2. Pick a point in the drawing.
  3. Property information is retrieved and displayed.
FLOODZONE -- FEMA Flood Zone Lookup
Queries the FEMA National Flood Hazard Layer (NFHL) in real time and returns the flood zone designation (AE, X, AH, etc.), base flood elevation (BFE), SFHA status, and FIRM panel number for any point you pick. Optionally places results as an MText label.
How to use:
  1. Run FLOODZONE.
  2. Pick a point in the drawing.
  3. Flood zone, BFE, SFHA status, and FIRM panel number are reported.
  4. Optionally place results as an MText label at the picked point.
FLOODCRITERIA -- County Flood Criteria
Queries the Miami-Dade County Flood Criteria 2022 dataset and reports the minimum finished floor elevation (ft NGVD) for the location you pick. Required for MDC development permits.
How to use:
  1. Run FLOODCRITERIA.
  2. Pick a point in the drawing.
  3. The Miami-Dade County minimum finished floor elevation is reported.
SECTIONLOOKUP -- PLSS Section Lookup
Queries the SFWMD PLSS service and returns the Section, Township, Range, and SSTTRR code for any point you pick. Required for legal descriptions and permit applications in Florida.
How to use:
  1. Run SECTIONLOOKUP.
  2. Pick a point in the drawing.
  3. Section, Township, Range, and SSTTRR code are reported.
GWMAY -- Water Table (Average May)
Queries the Miami-Dade County groundwater dataset and returns the average May water table elevation at the picked point in both NAVD 88 and NGVD 29. Used for seasonal high water table in stormwater and exfiltration design.
How to use:
  1. Run GWMAY.
  2. Pick a point in the drawing.
  3. Average May groundwater elevation is reported in NAVD 88 and NGVD 29.
GWOCT -- Water Table (October 2040 Projection)
Returns the projected October 2040 groundwater elevation at the picked point. Used for sea level rise resilience assessments and future-proofing drainage and foundation design in Miami-Dade County.
How to use:
  1. Run GWOCT.
  2. Pick a point in the drawing.
  3. Projected October 2040 groundwater elevation is reported in NAVD 88 and NGVD 29.

Areas & Excavation

AREAMANAGER -- Area Manager
A dockable palette for tracking impervious/pervious areas, lakes, and footprints across a project. Add areas by selecting hatches or closed polylines. Organize into categories, redraw deleted hatches, and export to CSV with per-category subtotals.
How to use:
  1. Run AREAMANAGER to open the palette.
  2. Create or select a project.
  3. Click Add Area and select a hatch or closed polyline.
  4. Name and categorize each area. Use Export to generate a CSV report.
EXF -- EXF Trench Manager
A dockable palette for tracking exfiltration trench lengths across a project. Select polylines or line segments representing trench runs; the tool accumulates total linear footage for permit submittals and contractor pay requests.
How to use:
  1. Run EXF to open the Trench Manager palette.
  2. Click Add Trench and select the polylines or lines representing trench runs.
  3. Total trench length is updated automatically.

As-Builts & Sections

SECDRAW -- Section Drawer
Opens a designer window where you build a cross-section by chaining slope segments (lane, shoulder, swale, cut/fill slope, etc.). Draws the section into model space at 1:1 scale with labeled dimensions.
How to use:
  1. Run SECDRAW.
  2. Add segments by entering width and slope for each element.
  3. Preview the section profile in the window.
  4. Click Draw and pick an insertion point in model space.
CORALASBUILT -- Coral Gables Sewer As-Built v1.1.0
Queries the Coral Gables Sewer GIS within 1000 ft of a picked point and draws gravity mains, force mains, manholes, and laterals into model space.
How to use:
  1. Run CORALASBUILT.
  2. Pick a center point in the drawing.
  3. The tool queries the Coral Gables Sewer ArcGIS FeatureServer and draws all sewer features within the specified radius directly into model space.
TABLEDRAW -- Table Drawer v1.1.0
Design Excel-like tables with merged cells, free text, and live links to Civil 3D entities. Draws a grid into model space and auto-updates when linked entities change.
How to use:
  1. Run TABLEDRAW.
  2. Define rows, columns, and cell content (free text or live links).
  3. Click Draw and select an insertion point.
  4. Linked cells will update automatically when the associated Civil 3D entity changes.

Annotation

CAD -- Linear dimensions between two reference lines + sub-dims for crossing lines v1.1.0
Pick two reference lines (at any angle) and an anchor point on line 1. The command builds a perpendicular measurement axis from that anchor through both lines, places one main AlignedDimension from line 1 to line 2 offset 5 drawing units perpendicular to the path, and 5 drawing units further out adds a sub-row of dimensions broken at every other line that crosses the path (line 1 -- obstacle 1 -- obstacle 2 -- ... -- line 2).
How to use:
  1. Run CAD.
  2. Pick the first reference line (line 1).
  3. Pick the second reference line (line 2).
  4. Pick an anchor point on line 1 -- the measurement path passes through this point perpendicular to line 1's direction.
  5. The main aligned dimension and the sub-row of obstacle dimensions are placed automatically.
Note: Obstacle detection considers Line entities in current-space that cross the measurement path strictly between the two reference lines. Polylines are picked segment-by-segment.

Vehicle Tracking

VTPANEL -- Vehicle Tracking Panel
The central hub for all vehicle tracking operations. Lists available vehicle templates, lets you configure vehicle parameters, and provides access to swept path, drive, and parking tools.
How to use:
  1. Run VTPANEL to open the Vehicle Tracking palette.
  2. Select a vehicle template from the list.
  3. Launch swept path, drive mode, or parking tools from the panel.
VTSWEEP -- Swept Path Analysis
Traces the selected vehicle along a polyline path and draws the swept envelope -- showing the full vehicle footprint through the maneuver. Essential for checking driveway geometry, intersection turning radii, and fire truck access.
How to use:
  1. Open VTPANEL and select a vehicle.
  2. Run VTSWEEP and draw or select the path polyline.
  3. The swept path envelope is drawn automatically.
VTDRIVE -- Interactive Drive Mode
Steer a vehicle through a maneuver in real time using the keyboard or mouse. The swept path is recorded and drawn as the vehicle moves. Useful for tight turning situations and fire access routes where the exact path is not predetermined.
How to use:
  1. Open VTPANEL and select a vehicle.
  2. Run VTDRIVE and click a starting position and direction.
  3. Use arrow keys or mouse to steer. The swept path draws in real time.
  4. Press Enter to finish and keep the path.
VTEDIT -- Edit Vehicle Path v1.1.0
Click on or near an existing Vehicle Tracking path or X marker to edit the maneuver. Hides the old geometry, shows a live jig preview, and redraws the updated path on finish.
How to use:
  1. Run VTEDIT.
  2. Click on or near a VT path or X marker.
  3. Adjust the waypoints and press Enter to save changes.
VTPARK -- Parking Layout Generator
Generates a parking layout -- stall lines and drive aisle -- for a selected vehicle type and configuration. Placed in model space at 1:1 scale. Speeds up preliminary parking design and ADA/zoning compliance checks.
How to use:
  1. Open VTPANEL and select a vehicle.
  2. Run VTPARK, configure stall angle and count.
  3. Pick an insertion point. The parking layout is generated with stall lines and aisle dimensions.

Quick Access

ALDTTOOLBAR -- Toggle Mini Toolbar
A compact floating panel with one-click access to every ALDT command. Stays on top of the drawing area and is organized by category with color-coded icons. Drag to reposition. Use the gear icon to show or hide individual commands.
How to use:
  1. Run ALDTTOOLBAR or click the Mini Toolbar button on the ALDT ribbon.
  2. Click any icon to run the corresponding command.
  3. Drag to reposition. Click X to close. Click the gear icon to customize visibility.
ALDTHELP -- Open Help Window
Opens this help document in a resizable window directly inside Civil 3D. All commands, workflows, and usage notes are available without leaving the application.
How to use:
  1. Run ALDTHELP from the ribbon or command line.
  2. The help window opens. Use the sidebar to jump to any command.
  3. Close the window when done.
